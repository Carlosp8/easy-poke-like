# Generated Pokelike Data

This directory is produced by `npm run extract:bundle`.

Tracked files here are normalized JSON snapshots extracted from the local
Pokelike bundle. Do not edit them by hand; copy a new bundle into the workspace
and rerun the extractor.

`bundle-meta.json` records:

- source bundle path and file name;
- SHA-256 hash;
- extraction timestamp;
- decoder details;
- section counts;
- warnings for missing expected sections.

Raw minified bundles are intentionally ignored by git.
